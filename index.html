JavaScript (JS)
A high-level, dynamic, untyped, and interpreted programming language. It has been standardized in the ECMAScript language specification.
jQuery
jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
Operator
A character that represents an action, such as multiplication (8) or division (/).
Variable
A storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.
Function
A named section of a program that performs a specific task.
Boolean
A data type with only two possible values: true or false
Method
A programmed procedure that is defined as part of a class and included in any object of that class. A class (and thus an object) can have more than one.
Parameter
Values that are passed to a function.
